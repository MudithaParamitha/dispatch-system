# Dispatch Letter Validation System

A web-based dashboard system for validating dispatch letters and tracking SPP status.

## Features
- Login authentication
- Dashboard interface
- SPP history table
- Status tracking (Approved / Pending / Rejected)
- Error handling pages (403, 404, 500)

## Tech Stack
HTML  
Bootstrap  
JavaScript  
DataTables
